# check
A simple script to run frequently used commands to ensure rust source code quality.
Additional flags can be seen with `check --help`.

## List of commands
- cargo check
- cargo fmt
- cargo clippy
- cargo test

## Required components
- rustc
- cargo
- clippy
- rustfmt

all of which can be installed with rustup.
